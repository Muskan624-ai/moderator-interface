# Moderator Interface - Frontend Task
A responsive admin interface for managing users and posts. Built with vanilla HTML, CSS, and JavaScript.

## Live Demo
https://adorable-bublanina-729307.netlify.app/

## Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox, Grid, Media Queries), JavaScript (ES6+).
* **API:** JSONPlaceholder.
* **Styling:** Custom CSS with responsive design.
* **Deployment:** Netlify & GitHub Pages.

## Key Features
* **User Management:** Displays a list of users with initials avatars.
* **Post Management:** Displays recent posts with the name of the users who posted them.
* **Responsive Design:** Optimized for mobile devices (collapsible sidebar layout).
* **UI/UX:**
    * **Compatible Colors:** High contrast orange & cream theme.
    * **Interactivity:** Hover effects, smooth transitions, and clickable user cards.
    * **Visuals:** Background pattern with transparency.
